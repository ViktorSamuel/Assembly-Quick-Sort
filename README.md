# Assembly-Quick-Sort
Quick sort implemented in assembly assembly 0x64 at&amp;t with C wrapper for input and output\
Compile using "gcc main.c sort.s -o prog"\
Run using "./prog"

## Exampe of usage:

### INPUT:
Number of elements: 3\
Element number 1: 5\
Element number 2: 8\
Element number 3: 0 

### OUTPUT:
Elements in sorted order: 0 5 8
